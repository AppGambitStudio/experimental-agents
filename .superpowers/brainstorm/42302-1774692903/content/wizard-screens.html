More thorough intake — collects all possible details upfront so the agent can run a comprehensive analysis without asking questions.

Step 1: Location — address, state, city
Step 2: Property — type, carpet area, floor, new/resale
Step 3: Financial — total price, bank/cash split, buyer gender, first property?
Step 4: Builder — name, RERA ID, GSTIN, seller name
Step 5: Land Records — district, taluka, village, survey no, document no

After Step 5 → runs full analysis automatically → shows report → switches to chat for Q&A.
