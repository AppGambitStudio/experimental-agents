Key Design Decisions

SSE for streaming
Tool calls, progress, and agent responses stream to the chat UI in real-time. User sees "Searching GujRERA..." as it happens.
Session persistence
Agent SDK sessions persist across messages. User can close browser, come back, and resume where they left off.
MCP stays in-process
All 3 MCP servers run inside the Hono process. No separate MCP server deployment. browser-mcp spawns dev-browser as needed.
One npm command
npm run dev starts both Vite (frontend) and Hono (backend). User opens localhost:5173 and starts using.
