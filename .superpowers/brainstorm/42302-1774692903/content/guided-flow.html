Guided Flow: Real Estate Agent

What style of guided experience should the property buyer get?

A

Step-by-Step Wizard

Multi-step form with one question per screen. Progress bar at top. Each step validates before moving forward. Final step shows "Analyzing..." with live progress, then the report.

Step 1 Step 2 Step 3 Step 4 Report
Where is the property?
Enter address or area name...
Example: "Vesu, Surat" or "Adajan, Near Science Centre"

Flow: Address → Property Type → Budget & Builder → RERA ID (optional) → Run Analysis → Report

B

Chat-Style Guided Conversation

The agent asks questions one at a time in a chat bubble format. User responds with text or clicks pre-filled options. Feels like talking to a consultant, not filling a form.

Flow: Conversational Q&A → Agent collects all inputs → Runs verification → Shows findings progressively in chat → User can ask follow-ups

C

Hybrid: Wizard Intake → Chat Follow-up

Quick 3-step wizard to collect essential info (address, type, budget). Then switches to chat mode for the interactive copilot experience — findings shown progressively, user can ask questions, use slash commands.
